# Student Performance Dashboard

## 📌 Project Overview
This project analyzes student academic performance using Excel dashboards and KPI reporting techniques.

The dashboard helps identify:
- High-risk students
- Attendance trends
- Student performance levels
- Pass rates and academic insights

---

## 📊 Features
- KPI Tracking
- Risk Level Analysis
- Attendance Analysis
- Student Performance Visualization
- Dashboard Reporting

---

## 🛠 Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Conditional Formatting
- GitHub

---

## 📈 KPIs Included
- Total Students
- Average Score
- High Risk Students
- Pass Rate
